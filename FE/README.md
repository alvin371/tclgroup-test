# FE Project

Simple frontend app built with React, TypeScript, and Vite.

## Requirements

- Node.js 20+
- npm 10+

## Installation

```bash
npm install
```

## Environment

Create `.env` from `.env.example`:

```bash
cp .env.example .env
```

Default values:

```bash
VITE_API_BASE_URL=/api/v1
VITE_DEV_API_PROXY_TARGET=http://localhost:8080
```

## Run

Start development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
